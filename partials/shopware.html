Shopware our plugin

Requirements

For the installation of the plugin, your system has to fulfill all requirements for Shopware. The plugin was developed for version 5.2.x of Shopware and in PHP version 5.6.27. Older versions of Shopware and PHP might be compatible with the plugin but are not supported. Please make sure to run the installation process in a test environment before updating your production system.

Installation and configuration

Installation

The following is a step-by-step instruction for the successful installation of the plugin:
  1. Login to the backend of your shop and select the plugin manager. The plugin manager is located under Configuration → Plugin Manager.
  2. Select Installed from the left menu bar and then Upload plugin from the top of the page to upload the zip file containing the plugin.
  3. The plugin Concardis Payengine should now appear under the tab Uninstalled in the left menu bar. Press the “+“ button to install the plugin.
  4. Following the installation process, the configuration page of the plugin is opened automatically. There you can adjust and activate the plugin.

Configure payment methods

All payment methods added through the plugin can be activated or deactivated in the backend of Shopware by selecting the menu item “Configuration → Payment methods”. Also, additional surcharges can be added. In its condition as supplied to customer, all payment methods are deactivated in the plugin.

Assign payment methods to shipping mode

Selecting the menu item Configuration → Shipping costs you can choose your preferred shipping mode and edit its configuration by clicking on the “pen” symbol. Switch to the tab Payment methods to assign payment methods to this shipping mode.

Hide payment methods

To hide the display of payment methods, choose to not activate them as shown under item 2.3 or select Configuration → Risk management to control the amount limit for not displaying the payment method. There, choose the payment method and then take the necessary adjustments.

Order processing

Within the order summary of Shopware (Customers → Orders) there is a detailed overview for every order which can be reached through the “pen” button. This overview shows the additional tab Concardis Payengine.

Overview

The tab Overview displays information involving the transaction, the payment method used, and the customer's data.

Transactions

The tab Transactions displays information about the transaction. Additionally, you can process the transaction through the buttons Capture, Cancel, or Refund. To Capture, Cancel, or Refund the transaction, select the transaction, displaying the articles associated with it in the section below. Then, the only thing displayed for this transaction will be the aforementioned buttons. If you want to process an article, for example by Capture, select the field under the tab “Selected” by clicking on it and enter the amount of the articles that need capturing. This process needs to be repeated for every button.

Capture

The collection of a reserved transaction can be achieved through pressing Capture.

Cancel

The cancellation of a transaction can be achieved through pressing Cancel.

Refund

The refund of a transaction can be achieved through pressing Refund.

Order modification

It is possible to modify an order* by adding, editing, or deleting articles. Select the “pen” icon of the order you would like to adjust. There, select the tab Concardis Payengine and subsequently “Positions”. *currently only applicable to orders with RatePAY payment method

Positions

To adjust the order, double-click on the articles to change them or press the “Add” button to add a new article. Pressing the “Confirm” button will update the order. Subsequently, the articles will be updated under Concardis Payengine → Transactions.
