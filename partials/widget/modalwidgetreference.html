Here you can find all needed methods and respective parameters to further enrich your checkout workflow.
! Important -> All methods should be called on PayEngineWidget object (e.g PayEngineWidget.desiredMethod(parameters)).
initAsModal
Initialize modal view with payment form. Always called on PayEngineWidget object.
Signature
initAsModal(publishableKey, orderId, optionalParameters, callback)
Arguments
Name Mandatory Type Default Value Description
publishableKey YES string N/A The MerchantID given to every user on registration in Merchant Center (can be found in My Account section)
orderId YES string N/A The ID of created order object
callback YES function N/A Function with following signature - callback (error, result)
optionalParameters.products NO Array inherited from order object Array with all supported payment methods. If paymentMethods is empty, null or undefined, all available payment methods for the widget are the ones, defined in order object. If paymentMethods is an array, it’s assumed that it’s a subset of payment methods array from the order object, so only defined by the integrator payment methods will be used. If there is inconsistency between the two arrays (e.g. integrator set more payment methods than defined in order object), error will be thrown.
optionalParameters.language NO string “en” The language used throughout the Widget. Available options are: "en" - English “de” - German
optionalParameters.redirectUser NO Boolean true Decides if end customer redirect (e.g. by PayPal payment) is handled by the Widget or manually. If true the Widget handles all redirects. If false redirects are handled manually by the integrator. ! In this case you should use the redirect URL given by the resultCallback.
optionalParameters.colorTheme NO string “light” Basic UI appearance. Available options are: “light” “dark”
optionalParameters.optionalFields.creditCard.cardHolder NO boolean false Is the "Card Holder" field mandatory for Credit Card.
optionalParameters.optionalFields.sepa.accountHolder NO boolean false Is the "Account Holder" field mandatory for SEPA.
optionalParameters.optionalFields.sepa.bic NO boolean false Is the "BIC" field mandatory for SEPA.
optionalParameters.optionalFields.sepa.bankName NO boolean false Is the "Bank Name" field mandatory for SEPA.
optionalParameters.optionalFields.ratepayDirectDebit.accountHolder NO boolean false Is the "Account Holder" field mandatory for Ratepay Direct Debit.
optionalParameters.optionalFields.ratepayDirectDebit.bic NO boolean false Is the "BIC" field mandatory for Ratepay Direct Debit.
optionalParameters.optionalFields.ratepayInstallment.accountHolder NO boolean false Is the "Account Holder" field mandatory for Ratepay Installment.
optionalParameters.optionalFields.ratepayInstallment.bic NO boolean false Is the "BIC" field mandatory for Ratepay Installment.
optionalParameters.deviceIdent NO boolean false Is Device Ident mandatory.
optionalParameters.paypalRecurringBillingAgreement NO boolean false Is Recurring Billing Agreement field mandatory for Paypal.
