RatePAY Reference


Here you can find all needed RatePAY helper functions.
appendDeviceIdent This method will append the RatePAY Device Ident code. Signature appendDeviceIdent (deviceIdentSId, deviceIdentToken) Arguments
Name Mandatory Type Default Value Description
deviceIdentSId YES String N/A DeviceIdentSId that is part of the ratepay merchant settings
deviceIdentToken YES String N/A Unique token generated by the backend of the integrator that is also to be sent along with the order capture request
getRatepayInstallmentConfig This method will return the RatePAY merchant configuration. Signature getRatepayInstallmentConfig (callback) Arguments
Name Mandatory Type Default Value Description
callback YES function N/A Function with following signature – callback (error, result) result parameter is object, containing the merchant configuration. error parameter is an object, containing a message with the reason what has failed. Please see Error object for further information.
getRatepayInstallmentCalculation This method will calculate an installment plan. Signature getRatepayInstallmentConfig(calculationRequest, callback) Arguments
Name Mandatory Type Default Value Description
calculationRequest YES Object N/A The calculation request hash. It should contain: amount: the amount to calculate an installment plan for. Mandatory calculationType: the plan can be calculated based on duration ('calculation-time') or monthly rate ('calculation-rate'). Mandatory. calculationTime: number of months to calculate the plan for. Mandatory if type is time. calculationRate: amount of monthly rate to calculate the plan for. Mandatory if type is rate.
callback YES function N/A Function with following signature – callback (error, result) result parameter is object, containing the calculation result. error parameter is an object, containing a message with the reason what has failed. Please see Error object for further information.
