Common Reference


Here you can find all methods and variables common for every BridgeJS solution. ! Important -> All methods should be called on PayEngine object (e.g PayEngine.desiredMethod(parameters)).
setPublishableKey Identifies the merchant against the Bridge API, based on the given publishableKey (Merchant ID). Signature setPublishableKey (publishableKey) Arguments
Name Mandatory Type Default Value Description
publishableKey YES String N/A The MerchantID given to every user on registration in Merchant Center (can be found in My Account section)
Error object Can be returned in any callback function as the error parameter. Signature 
{
error: {General error message},
status: {General status code},
message: {
 properties: [
 key: {Error details key},
 code: {Error details code},
 message: {Error details message}
 ]
}
}
